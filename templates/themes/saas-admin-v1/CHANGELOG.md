# SoftDesk — Changelog

## v1.0.0 (April 2026)

### Initial Release

**Pages:**
- Dashboard — KPI stat cards, revenue line chart, subscriber bar chart, plan doughnut, activity feed
- Users — Searchable data table with filters, sort, pagination, stat cards
- User Detail — Profile card, subscription info, download history, activity timeline
- Billing — Plan comparison cards (4 plans), transaction table with status badges
- Settings — Profile form, notification toggles, security (password change), API keys, danger zone

**Design System:**
- Neumorphism (Light) with premium-grade rgba shadows
- Light reflection borders (top/left 1px highlight)
- 4 accent color presets: Indigo-Violet, Ocean Blue, Emerald, Coral Rose
- Inter font, 16px body, tabular-nums for data
- 8px spacing grid, 12/16/20/24px radius system

**Accessibility:**
- WCAG AA color contrast throughout
- Keyboard navigation support
- ARIA labels on all interactive elements
- Semantic HTML5 structure

**Responsive:**
- Desktop (1280px+): Full sidebar
- Tablet (768–1279px): Collapsed icon sidebar
- Mobile (–767px): Bottom navigation
