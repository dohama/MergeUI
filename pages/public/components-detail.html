Stat Card

Free

A compact card displaying a single KPI metric with label, value, and trend indicator. Perfect for dashboard overviews.

Preview

Code

<div class="stat-card">
  <span class="stat-label">Total Revenue</span>
  <span class="stat-value">$48,290</span>
  <span class="stat-trend positive">
    +12.5% <span class="trend-note">vs last month</span>
  </span>
</div>
.stat-card {
  background: var(--merge-bg-surface, #16161E);
  border: 1px solid var(--merge-border, rgba(255,255,255,0.06));
  border-radius: 14px;
  padding: 20px 24px;
  display: flex;
  flex-direction: column;
  gap: 4px;
}
.stat-label {
  font-size: 12px;
  font-weight: 500;
  color: var(--merge-text-muted, #71717A);
}
.stat-value {
  font-size: 28px;
  font-weight: 800;
  color: var(--merge-text-primary, #F4F4F5);
  letter-spacing: -1px;
}
.stat-trend {
  font-size: 12px;
  font-weight: 600;
}
.stat-trend.positive { color: var(--merge-status-success, #22C55E); }
.stat-trend.negative { color: var(--merge-status-error, #EF4444); }
.trend-note {
  color: var(--merge-text-muted, #71717A);
  font-weight: 400;
}
