## 컨셉

- **스타일**: Claymorphism (3D 둥글둥글, 부드러운 컬러, 내부 그림자)
- **카테고리**: Finance (포트폴리오, 거래, 수입/지출, 예산)
- **분위기**: 안정감 + 고급 (에메랄드+골드)
- **레이아웃**: 미니 사이드바 72px (아이콘 + 호버 시 확장)

## 클레이모피즘 스타일

```css
/* 클레이 카드 — 3D 둥글둥글 */
.clay-card {
  background: #FFFFFF;
  border-radius: 20px;
  box-shadow:
    8px 8px 16px rgba(0,0,0,0.08),
    inset -2px -2px 4px rgba(0,0,0,0.03),
    inset 2px 2px 4px rgba(255,255,255,0.8);

/* 클레이 카드 호버 */
.clay-card:hover {
  transform: translateY(-3px);
  box-shadow:
    10px 10px 20px rgba(0,0,0,0.10),
    inset -2px -2px 4px rgba(0,0,0,0.03),
    inset 2px 2px 4px rgba(255,255,255,0.8);
}

/* 클레이 버튼 */
.clay-btn {
  background: linear-gradient(145deg, #00B894, #00A381);
  border-radius: 14px;
  box-shadow:

    inset -1px -1px 2px rgba(0,0,0,0.1),
    inset 1px 1px 2px rgba(255,255,255,0.2);
```

## 레이아웃 — 미니 사이드바

```
┌──┬───────────────────────────────────┐
│  │                                   │
│72│     Topbar (60px)                 │
│px│───────────────────────────────────│
│  │                                   │
│아│     Main Content                  │
│이│     (패딩 32px)                    │
│콘│                                   │
│  │                                   │
└──┴───────────────────────────────────┘

호버 시 사이드바 → 240px 확장 (오버레이)
```

- 기본: 72px 아이콘만
- 호버/클릭: 240px로 확장 (오버레이, 콘텐츠 밀지 않음)
- 모바일: 숨김 → 하단 네비

## 페이지 구성 (5페이지)

| 페이지 | 실제 금융 항목 |
|--------|--------------|
| Overview | 총 자산, 수입/지출 비교 차트, 최근 거래, 예산 진행률 |
| Transactions | 거래 내역 테이블, 카테고리 필터, 날짜 범위, 검색 |
| Portfolio | 투자 포트폴리오, 자산 배분 도넛, 수익률 차트 |
| Budget | 카테고리별 예산 vs 실제, 진행률 바, 월별 비교 |
| Reports | 수입/지출 추이, 카테고리 분석, 연간 요약 |
