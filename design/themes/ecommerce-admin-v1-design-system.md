# E-Commerce Admin v1 — Design System

> 글래스모피즘 이커머스 대시보드
> B(디자인 디렉터) | 2026-04-18

---

## 컨셉

- **스타일**: Glassmorphism (반투명 유리 효과, 블러 배경, 미세 보더)
- **카테고리**: E-Commerce (주문, 상품, 재고, 고객, 매출)
- **분위기**: 활기찬 + 프리미엄 (코랄+오렌지 그라디언트)
- **레이아웃**: 탑바 Only (사이드바 없음, 가로 메뉴) — 기존 테마와 차별화

## 타이포그래피

- **메인 폰트**: DM Sans (Google Fonts, OFL)
- 깔끔한 지오메트릭 폰트, 이커머스의 모던한 느낌에 적합

| 요소 | 크기 | 굵기 |
|------|------|------|
| 페이지 타이틀 | 24px | 700 |
| 카드 타이틀 | 16px | 600 |
| Body | 14px | 400 |
| KPI Value | 32px | 700 |
| Caption | 12px | 500 |

## 글래스모피즘 스타일

```css
/* 유리 카드 */
.glass-card {
  background: rgba(255, 255, 255, 0.06);
  backdrop-filter: blur(20px);
  -webkit-backdrop-filter: blur(20px);
  border: 1px solid rgba(255, 255, 255, 0.12);
  border-radius: 16px;
}

/* 유리 카드 호버 */
.glass-card:hover {
  background: rgba(255, 255, 255, 0.10);
  border-color: rgba(255, 255, 255, 0.18);
}
```

## 레이아웃 — 탑바 Only

```
┌──────────────────────────────────────────┐
│ Logo  │  메뉴1  메뉴2  메뉴3  │  검색  아바타 │  ← 탑바 (72px)
├──────────────────────────────────────────┤
│                                          │
│          Main Content Area               │
│          (max-width: 1440px)             │
│          (패딩 32px)                      │
│                                          │
└──────────────────────────────────────────┘
```

- 사이드바 없음 → 콘텐츠 영역 넓음
- 탑바에 가로 메뉴
- 모바일: 햄버거 메뉴

## 페이지 구성 (5페이지)

| 페이지 | 실제 이커머스 항목 |
|--------|-------------------|
| Overview | 매출 KPI, 주문 추이 차트, 인기 상품 TOP 5, 최근 주문 |
| Orders | 주문 목록 테이블, 상태 필터 (처리중/배송중/완료/취소), 검색 |
| Products | 상품 카드 그리드, 카테고리 필터, 재고 상태 |
| Customers | 고객 목록, 가입일/주문수/총 구매금액 |
| Analytics | 매출 차트, 카테고리별 매출, 지역별 분포, 리텐션 |
