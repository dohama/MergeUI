## 2. Launch Description (영문, 300자 이내)

MergeUi gives you production-ready dashboard templates built with pure HTML & CSS. Dark + light mode, fully responsive, 50+ components. No framework lock-in — copy into React, Vue, Svelte, or plain HTML. Change CSS variables, ship in minutes. Free tier available.

## 3. Maker Comment Draft (영문)

Hey Product Hunt! I'm the maker of MergeUi.

I kept rebuilding the same dashboard UI for every project — sidebar, stat cards, charts, tables. Every time, 2+ weeks gone on pixels instead of product logic.

MergeUi is what I wish existed: production-ready dashboard templates that are framework-agnostic, dark mode included, and customizable through CSS variables alone. No npm install, no build step, no dependency hell. Just copy the HTML/CSS into your project and change the variables.

We have a free tier so you can try it right now. Would love your feedback on what components or themes you'd want next.
