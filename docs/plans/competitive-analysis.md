## 1. Tailwind UI (현재 "Tailwind Plus")

**가격 모델**: 일회성 결제 $299 (약 40만원) -- 평생 접근권
- 한 번 사면 모든 컴포넌트 + 템플릿 + 향후 추가분까지 영구 이용
- 추가 프로젝트에 재구매 불필요

**제공 범위**:
- 500개 이상의 컴포넌트
- React/Next.js 기반 사이트 템플릿
- 반응형, 다크모드 지원

**강점**:
- Tailwind CSS 공식 제품이라는 브랜드 신뢰도
- 코드 품질이 업계 최고 수준
- 일회성 결제 = 구매 허들이 낮음 (구독 피로 없음)
- 지속적인 신규 콘텐츠 추가

**약점**:
- 대시보드/어드민 전용이 아님 -- 마케팅 페이지, 이커머스 등 범용
- React/Next.js에 묶여 있음 -- 순수 HTML/CSS 사용자는 접근성 낮음
- $299는 개인 개발자에게 부담될 수 있음
- 커스터마이징 가이드가 부족 (코드를 줄 뿐, 어떻게 바꾸라는 안내가 약함)

**MergeUi가 배울 점**:
- "평생 접근" 모델의 심리적 매력 (구독 피로 제거)
- 신규 콘텐츠를 꾸준히 추가하여 입소문 유지

**MergeUi 차별화 포인트**:
- 대시보드/어드민에 특화 --> "대시보드가 필요하면 MergeUi"라는 명확한 포지션
- 순수 HTML/CSS 제공 --> 프레임워크에 안 묶임, 어떤 프로젝트든 적용 가능
- 구독 모델로 지속적 업데이트 제공 + Free 플랜으로 진입 허들 낮춤

## 2. shadcn/ui

**가격 모델**: 완전 무료 (오픈소스)
- 컴포넌트를 복사해서 자기 프로젝트에 붙여넣는 방식
- CLI 도구로 설치 가능

**제공 범위**:
- 40개 이상의 기본 UI 컴포넌트 (버튼, 모달, 폼 등)
- Radix UI + Tailwind CSS 기반
- GitHub 스타 90,000개 이상, 주간 npm 다운로드 250,000회 이상

**강점**:
- 무료 = 진입 장벽 제로
- "코드를 소유한다"는 철학 --> 개발자들이 열광
- AI 도구(v0.dev 등)와 궁합이 좋음
- React/Next.js 생태계의 사실상 표준

**약점**:
- 대시보드 전용이 아님 -- 기본 UI 컴포넌트 수준
- 차트, 테이블, 사이드바 같은 대시보드 핵심 컴포넌트가 부족
- 디자인이 미니멀 --> 비개발자가 보기에 "완성품" 느낌이 약함
- React 전용 --> 다른 프레임워크 사용자는 쓸 수 없음
- 무료라서 수익 모델 자체가 없음

**MergeUi가 배울 점**:
- "코드 소유" 철학 --> MergeUi도 구독자가 코드를 100% 수정 가능하도록
- 개발자 커뮤니티 마케팅의 위력 (GitHub, Twitter, Dev.to)

**MergeUi 차별화 포인트**:
- shadcn은 "레고 블록"이고 MergeUi는 "완성된 집" --> 바로 쓸 수 있는 대시보드
- 차트, 복잡한 테이블, 위젯 등 대시보드 전문 컴포넌트 제공
- 라이트/다크 모드가 처음부터 완벽하게 디자인된 상태로 제공
- 프레임워크 독립적 (HTML/CSS)

## 3. Tremor

**가격 모델**: 완전 무료 (오픈소스, Apache-2.0/MIT 라이선스)
- 2025년 1월 Vercel에 인수됨
- 인수 후 유료였던 Tremor Blocks도 전부 무료 공개

**제공 범위**:
- React 전용 차트/대시보드 컴포넌트
- 라인 차트, 바 차트, 도넛 차트 등 데이터 시각화 특화
- Tailwind CSS 기반

**강점**:
- 대시보드/데이터 시각화에 특화된 몇 안 되는 라이브러리
- Vercel 인수 후 장기 지원 보장
- 무료 = 진입 장벽 없음
- 차트 컴포넌트 품질이 높음

**약점**:
- React 전용 -- 다른 프레임워크 불가
- "완성된 대시보드 테마"가 아니라 "컴포넌트 모음"
- 레이아웃, 사이드바, 네비게이션 등은 직접 만들어야 함
- Vercel 생태계에 묶일 가능성

**MergeUi가 배울 점**:
- 차트/데이터 시각화의 디자인 퀄리티 벤치마크
- "대시보드 특화"라는 포지셔닝의 유효성 확인

**MergeUi 차별화 포인트**:
- Tremor는 "차트 컴포넌트"이고 MergeUi는 "완성된 대시보드 페이지"
- 프레임워크 독립적 (HTML/CSS)
- 레이아웃 + 네비게이션 + 차트 + 테이블이 모두 통합된 완성품

## 4. AdminLTE / CoreUI / Tabler

**가격 모델**:
- AdminLTE: 완전 무료 (MIT 라이선스)
- CoreUI: 무료 버전 + Pro $127/년
- Tabler: 무료 버전 + Pro $29

**제공 범위**:
- 완성된 어드민/대시보드 템플릿
- Bootstrap 기반 (AdminLTE, CoreUI)
- 다양한 프레임워크 지원 (CoreUI: Bootstrap, Angular, React, Vue)

**강점**:
- AdminLTE는 GitHub 스타 44,000개 이상, 역사가 긴 검증된 제품
- 무료로도 충분히 쓸 만한 수준
- Bootstrap 생태계의 방대한 사용자 기반

**약점**:
- 디자인이 구식 -- 2020년대 초반 느낌에서 크게 발전하지 못함
- Bootstrap 의존 --> Tailwind 시대에 뒤처지는 느낌
- 업데이트 주기가 느림
- "고급스러운" 느낌이 부족 --> 유료로 팔기엔 프리미엄감이 약함
- 커스터마이징이 번거로움

**MergeUi가 배울 점**:
- 무료 버전의 완성도가 높으면 유료 전환에 도움 (AdminLTE의 인지도)
- 다중 프레임워크 지원의 가치 (CoreUI)

**MergeUi 차별화 포인트**:
- 현대적 디자인 (2026년 트렌드 반영, 글래스모피즘, 미니멀리즘)
- Tailwind CSS 기반의 깔끔한 코드
- 라이트/다크 모드가 단순 색 반전이 아닌 재설계
- CSS 변수 기반 커스터마이징 --> 변수만 바꾸면 전체 테마 변경
- 정기 업데이트로 항상 최신 트렌드 반영

## 5. Envato / ThemeForest (마켓플레이스)

**가격 모델**: 마켓플레이스 (개별 템플릿 판매)
- 대시보드 템플릿 가격대: $10~$59 (일회성)
- 작가 수수료: 비독점 55%, 독점 12.5~37.5%
- 구매자에게 별도 수수료 부과

**제공 범위**:
- 2,974개 이상의 어드민 대시보드 템플릿
- HTML, React, Angular, Vue 등 다양한 기술 스택
- 개별 작가들이 올리는 마켓플레이스 구조

**강점**:
- 어마어마한 선택지 (거의 3,000개 템플릿)
- 개발자들이 이미 잘 아는 플랫폼
- 리뷰/평점 시스템으로 품질 검증
- 일회성 결제 = 구매 부담 낮음

**약점**:
- 품질 편차가 심함 -- 좋은 것도 있고 나쁜 것도 많음
- 구매 후 업데이트가 불규칙 (작가가 관리 안 하면 방치)
- 수수료가 높아서 작가 수익이 적음 --> 고급 작가 이탈
- 템플릿 간 디자인 일관성 없음 (각 작가 스타일이 다름)
- 코드 품질 보장이 안 됨
- 고객 지원이 작가 개인에 의존

**MergeUi가 배울 점**:
- 라이브 데모의 중요성 (ThemeForest 베스트셀러는 모두 라이브 데모 제공)
- 상세한 스크린샷과 기능 목록이 전환율(구매로 이어지는 비율)을 높임

**MergeUi 차별화 포인트**:
- 일관된 디자인 시스템 --> 모든 테마가 같은 디자인 토큰을 공유
- 구독 모델 --> 항상 최신 업데이트를 받을 수 있음
- 단일 브랜드의 품질 보증 --> ThemeForest의 품질 편차 문제 해결
- CSS 변수 기반 커스터마이징 체계 --> "변수만 바꾸면 됩니다"

## 전략 권고

### MergeUi의 핵심 포지션: "대시보드 전문, 프레임워크 독립, 구독형 디자인 시스템"

**핵심 차별화 3가지:**

1. **"완성품" 전략**: shadcn/Tremor는 조립이 필요하고, ThemeForest는 품질이 들쭉날쭉. MergeUi는 "열면 바로 쓰는 완성된 대시보드"
2. **프레임워크 독립**: 경쟁사 대부분이 React에 묶여 있음. MergeUi는 순수 HTML/CSS로 어디서든 사용 가능
3. **CSS 변수 커스터마이징**: "변수 10개만 바꾸면 내 브랜드 대시보드 완성" --> 5분 내 적용이라는 가치 제안

**가격 전략 권고:**
- Free: 테마 1개 미리보기 + 기본 컴포넌트 3~5개 (맛보기)
- Pro: $19/월 또는 $149/년 --> 전체 테마 + 컴포넌트 + 업데이트 (Tailwind UI $299 대비 진입 장벽 낮춤)
- Team: $49/월 또는 $399/년 --> 5인 라이선스

> Tailwind UI가 $299 일회성인 점을 감안하면, 연간 $149는 "매달 새 테마가 추가되니까" 라는 가치로 정당화 가능
