# Maison Finance (lx_v1)

> European Luxury · 프라이빗 뱅킹 · Wealth Management 대시보드
> 디자인 스펙: [`design/themes/lx_v1-design-system.md`](../../design/themes/lx_v1-design-system.md)

## 시그니처
- **이중 배경**: 크림 사이드바 `#F5F0E8` + 딥 차콜 메인 `#1A1815`
- **앤틱 골드 `#C9A961`** 액센트 (과용 금지, 포인트만)
- **세리프 3종 혼용**: Playfair Display (H1·KPI) + Cormorant Garamond (italic 포인트) + Montserrat (본문)
- KPI 값은 36px Playfair Display italic (세리프 숫자)
- **L-shape 골드 코너 장식** (카드 좌상·우하단)
- **··· 점 3개 디바이더** (섹션 구분)
- 센터 심메트릭 탑바 (letter-spacing 0.3em)
- Border-radius 0 또는 2/4px (샤프한 클래식)
- 아웃라인 버튼 → 호버 시 골드 채움

## Quick Start
1. `index.html` 브라우저 열기
2. 또는 `npx serve`

## 커스터마이징

- **데이터**: `js/data.js`의 `LX_DATA`
