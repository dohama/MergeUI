## GitHub README

```
# MergeUi

Production-ready dashboard templates for developers.

- **Framework-agnostic** — Pure HTML & CSS. Use with React, Vue, Svelte, Angular, or vanilla.
- **Dark + Light mode** — Every template ships with both themes.
- **50+ components** — Charts, tables, cards, forms, modals, sidebars. Copy-paste ready.
- **CSS Variables** — All design tokens as custom properties. Change your brand in one file.
- **Responsive** — Desktop, tablet, mobile.
- **Accessible** — WCAG AA, keyboard navigation, ARIA labels.

## Quick Start

1. Download a theme
2. Open the HTML file
3. Change `--merge-color-primary` to your brand color
4. Ship it

## Pricing

- **Free**: 1 theme, 5 components
- **Pro** ($19/mo): All themes, 50+ components, monthly updates
- **Team** ($49/mo): Pro + 10 seats, team license

[mergeui.com](https://mergeui.com)
```
