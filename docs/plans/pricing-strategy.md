## 1. 에이전트별 분석 요약

### A(PM) — 비즈니스 모델 분석

**구독 vs 일회성 vs 하이브리드:**

| 모델 | 장점 | 단점 | MergeUi 적합도 |
|------|------|------|----------------|
| 일회성 ($299, Tailwind UI 방식) | 구매 허들 낮음, 구독 피로 없음 | 반복 매출 없음, 성장에 신규 고객만 의존 | 낮음 |
| 구독 ($19~49/월) | MRR(월간 반복 매출) 예측 가능, LTV 높음 | 이탈률 관리 필요, "매달 뭐가 새로 나오나" 기대 관리 | 높음 |
| **하이브리드 (구독 + 단품)** | 양쪽 장점 결합, 진입 경로 다양화 | 가격 구조 복잡해질 수 있음 | **최적** |

**결론: 하이브리드 권장** — 구독을 메인으로 하되, 테마 단품 구매도 제공하여 "구독까지는 아직..." 하는 개발자도 잡는다.

**LTV/CAC 기반 적정 가격:**
- 목표 LTV:CAC 비율: 3:1 이상
- 개발자 대상 CAC(고객 획득 비용): 콘텐츠 마케팅 중심이면 $15~30 수준
- 목표 LTV(고객 생애 가치): $90 이상
- 평균 구독 유지 기간: 8~12개월 (SaaS 업계 중위값)
- **역산**: LTV $90 / 8개월 = 월 $11.25 → 마진 고려 시 월 $29 이상이 안정적

**이탈률(Churn) 가정:**
- 월간 이탈률 목표: 5% 이하 (연간 45% 이탈)
- 이탈 방지 핵심: 매월 신규 테마/컴포넌트 추가 + 기존 테마 업데이트

### B(디자인) — 제품 가치 평가

**디자이너가 직접 대시보드를 만들면?**
- 시니어 UI 디자이너 일당: $400~600 (한국 기준 50~80만원)
- 대시보드 테마 1개 디자인 공수: 5~8일 (레이아웃 + 컴포넌트 + 다크모드 + 반응형)
- **디자인만 $2,000~4,800** (코딩 별도)
- 디자인 시스템/토큰 구축 공수: 3~5일 추가 → **$1,200~3,000**
- MergeUi 테마 1개의 디자인 가치: **$3,200~7,800**

**컴포넌트 50개의 가치:**
- 개별 컴포넌트 디자인: 평균 0.5일 × $500 = $250/개
- 50개 × $250 = **$12,500**
- 일관된 디자인 시스템 안에서의 통합 가치: 개별 합산 대비 1.5배 → **$18,750**

**결론:** MergeUi가 제공하는 디자인 가치는 직접 의뢰 시 **$20,000 이상**. 연간 $200~400은 극히 합리적인 가격.

### C(프론트엔드) — 개발자 편의성 가치

**개발자가 직접 대시보드를 코딩하면?**
- 시니어 프론트엔드 개발자 일당: $400~800
- 대시보드 테마 1개 코딩 공수: 7~12일 (HTML/CSS/JS + 반응형 + 다크모드 + 접근성)
- **코딩만 $2,800~9,600**
- 차트/테이블/사이드바 등 복잡한 컴포넌트: 추가 5~10일

**MergeUi의 개발자 편의성 가치:**

| 항목 | 가치 설명 | 가격 정당화 |
|------|----------|------------|
| 5분 내 적용 | 다운로드 → 복붙 → CSS 변수 수정 → 끝 | 개발 시간 최소 40시간 절약 |
| 프레임워크 독립 | React/Vue/Svelte/바닐라 어디서든 사용 | 기술 부채 없음 |
| CSS 변수 커스터마이징 | 변수 10개만 바꾸면 브랜드 적용 완료 | 커스터마이징 공수 90% 절감 |
| 시맨틱 HTML + 접근성 | ARIA, 키보드 네비게이션 기본 탑재 | 접근성 구현 공수 별도 절약 |
| 코드 품질/문서 | 깨끗한 코드 + 상세 가이드 | 유지보수 비용 절감 |

**프레임워크별 래퍼 제공에 따른 가치 차이:**
- 순수 HTML/CSS만 제공 (현재): Pro 기준 적정
- React/Vue/Svelte 래퍼 추가 제공: 가격 30~50% 프리미엄 가능
- **권장**: MVP 단계에서는 HTML/CSS에 집중, Scale-up에서 React 래퍼 추가 시 가격 인상 근거로 활용

### D(백엔드) — 라이선스/배포 모델

**라이선스 범위:**

| 라이선스 | 적용 범위 | 가격 반영 |
|---------|----------|----------|
| 개인(Personal) | 본인 프로젝트 1개 | 기본가 |
| 상용(Commercial) | 고객 납품 프로젝트 포함, 무제한 | 1.5~2배 |
| 팀(Team) | 5인까지 동일 라이선스 공유 | 2.5~3배 |
| 엔터프라이즈 | 무제한 인원, SaaS 재판매 가능 | 별도 협의 |

**구독 해지 후 정책:**
- 구독 중 다운로드한 파일: **영구 사용 가능** (업계 표준)
- 구독 해지 후: 신규 테마/컴포넌트/업데이트 접근 차단
- 이 정책이 구독 유지 동기 부여 + 해지 허들 낮춤 (= 가입 허들도 낮춤)

**기술적 배포 방식:**
- MVP: ZIP 다운로드 (Lemonsqueezy 파일 호스팅)
- Scale-up: CDN 기반 핫링크 + npm 패키지
- Vision: CLI 도구 (`npx mergeui init`)

### E(QA) — 품질 보증 가치

**품질 보증이 가격에 미치는 영향:**
- 크로스브라우저 테스트 (Chrome/Firefox/Safari/Edge): ThemeForest 템플릿 대비 확실한 차별화
- 반응형 3단계 검증 (데스크톱/태블릿/모바일): 직접 만들면 QA만 2~3일
- WCAG AA 접근성 보장: 접근성 감사 외주 시 $500~2,000
- **매 릴리즈 15점 체크리스트 통과**: "MergeUi는 품질이 보장된다"는 신뢰 = 프리미엄 가격 정당화

**업데이트/버그픽스 포함:**
- 구독 기간 중 모든 버그픽스 무료 (당연)
- 브라우저 업데이트로 인한 호환성 문제 즉시 대응
- 이것이 일회성 구매 대비 구독의 핵심 가치

### F(그로스) — 시장/전환 분석

**경쟁사 가격 상세 비교:**

| 제품 | 가격 | 모델 | 포함 범위 |
|------|------|------|----------|
| Tailwind UI (Plus) | $299 일회성 | 평생 접근 | 500+ 컴포넌트, 사이트 템플릿 (대시보드 전용 아님) |
| Creative Tim Pro | $39~89/템플릿 | 일회성 | 단일 템플릿 + 6개월 지원 |
| CoreUI Pro | $127/년 | 연간 구독 | 어드민 템플릿 + 컴포넌트 (Bootstrap 기반) |
| Tabler Pro | $29 일회성 | 평생 접근 | 어드민 대시보드 (디자인 평범) |
| ThemeForest 대시보드 | $10~59 일회성 | 단일 구매 | 품질 편차 심함 |
| Apex/Flux (최신) | $69 일회성 | 단일 구매 | Next.js + shadcn 기반 |
| **MergeUi (제안)** | **$29/월 또는 $199/년** | **구독** | **전체 테마 + 컴포넌트 + 매월 업데이트** |

**개발자 지불 의향(WTP) 분석:**
- 개발자 도구 시장에서 월 $20~50 구독은 "생산성 도구" 범주 (GitHub Copilot $19, Vercel Pro $20)
- 대시보드 템플릿은 "한 번 사면 끝" 인식이 강함 → 구독 정당화를 위해 "매월 새 콘텐츠"가 핵심
- $29/월은 "점심 한 끼 + 커피 한 잔" 수준 → 의사결정 부담 낮음
- $199/년은 "Tailwind UI $299보다 $100 저렴하면서 매달 새 것이 추가됨" → 명확한 앵커링(가격 비교 기준점)

**가격 앵커링 전략:**
- 랜딩페이지에 "직접 만들면 $20,000+" vs "MergeUi Pro 연간 $199" 비교 배치
- Team 플랜을 가운데에 배치하여 Pro가 "합리적"으로 보이게 (디코이 효과)
- 연간 결제 시 월 환산 가격 강조 ("월 $16.6부터")

## 3. 최종 권고: 2안 (Scale-up) 채택

**이유:**
1. **가격 앵커링이 명확**: Pro 연간 $199 vs Tailwind UI $299 → "더 저렴하면서 매달 새 것이 추가"
2. **캡틴의 디자인 퀄리티 반영**: $29/월은 "고급 대시보드 디자인 시스템"에 합당
3. **얼리버드로 초기 저항 해소**: 런칭 가격은 $19/월 수준으로 시작 (아래 이벤트 참조)
4. **성장 여지**: 콘텐츠가 쌓이면 3안(Vision)으로 자연스럽게 전환 가능

## 5. 일회성 상품 (테마 단품 판매)

**구독이 부담스러운 개발자를 위한 단품 옵션:**

| 상품 | 가격 | 포함 내용 |
|------|------|----------|
| 테마 단품 | **$49/테마** | 테마 1개 + 해당 테마 컴포넌트 + 다크모드 |
| 테마 번들 (3개) | **$119** (개별 대비 19% 할인) | 테마 3개 + 전체 컴포넌트 |
| 컴포넌트 팩 | **$29** | 컴포넌트 전체 (테마 미포함) |

**단품 vs 구독 비교 (구독 유도용):**
- 테마 단품 $49 × 3개 = $147 vs Pro 연간 $199 (매월 신규 추가 포함)
- "3개 이상 필요하면 구독이 이득" → 자연스러운 업셀링

**라이선스:** 단품도 상용 사용 가능, 단 업데이트는 구매 시점 버전까지만
